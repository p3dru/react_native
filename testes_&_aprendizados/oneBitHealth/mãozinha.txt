1 - Instalação
    1.1 - É preciso instalar o node. primeiramente
    1.2 - Com o node instalado, instalamos o expo com o comando (no cmd): npm install --global expo-cli
    1.3 - Para criar o projeto: expo init nome_do_projeto
    1.4 - Selecionar a opção blank
    1.5 - Acessar a pasta criada: cd nome_do_projeto
    1.6 - Na pasta acessada, para iniciar o projeto: npm start
    1.7 - Daí é só escanear o QRcode e sair testando
